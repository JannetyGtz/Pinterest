== README

Aplicacion realizada en Ruby on rails, en un ejemplo de la red social Pinterest.
Puedes crear usuarios, crear pins juntos con su posible edicion y puedes eliminarlos.
Tambien puedes tener acceso para ver los pins de las demas personas.
Se utiliza Bootstrap para complementar el diseno por medio de CSS.


Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
